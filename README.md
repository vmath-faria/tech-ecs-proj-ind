# tech-ecs-proj-ind
Olá, Usuário!

Este é um projeto individual, com tema centrado no legado do BLACK SABBATH.

Criados em 1968, sua carreira de quase 6 décadas os pôs firmemente no topo do panteão das lendas da Música, sendo a banda que fundou o Heavy Metal. 

Este projeto tem como objetivo mostrar a história da banda, oferecer um quiz para os usuários, testando seus conhecimentos e identificando suas preferências musicais, além de fornecer dados coletados na pesquisa por meio de dashboards e KPIs.
